  # 颐尔福驻车空调售后反馈小程序

## 项目简介

这是一个专为驻车空调用户设计的售后问题反馈平台，支持客户、主管、维修工三种角色的差异化操作。

## 功能特性

### 客户端功能
- **问题反馈**：支持电话反馈和表单反馈两种方式
- **进度查询**：实时查看问题处理进度
- **FAQ列表**：常见问题解答和搜索功能

### 主管端功能
- **待处理问题列表**：查看新提交的问题，支持优先级设置和分配
- **已分配列表**：跟踪已分配问题的处理进度
- **已解决列表**：查看已完成的问题
- **数据统计**：问题类型分布、处理时效、维修工工作量统计

### 维修工端功能
- **我的任务列表**：按优先级排序的待处理任务
- **任务详情**：处理记录、完成按钮、协助申请
- **历史记录**：已完成任务的历史记录

## 技术架构

- **框架**：微信小程序原生框架
- **语言**：JavaScript、WXML、WXSS
- **设计风格**：简洁高效，主色调蓝色+橙色
- **组件化**：可复用的UI组件

## 核心功能实现

### 1. 角色权限管理
- 基于角色的页面访问控制
- 动态TabBar配置
- 权限验证装饰器

### 2. 表单验证
- 手机号、邮箱、密码强度验证
- 文件类型和大小验证
- 自定义验证规则

### 3. 网络请求封装
- 统一的请求拦截器
- 自动错误处理
- 文件上传下载支持

### 4. 组件化设计
- 可复用的UI组件
- 统一的样式规范
- 响应式布局
