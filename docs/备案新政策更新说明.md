# 微信小程序备案完整指南（2025年）

## 📅 文档更新：2025年10月25日

---

## 📌 重要说明

本文档整理了微信小程序备案的**现行要求**。以下内容为2024年4月起正式实施的政策，目前（2025年）已全面执行。

---

## 📋 当前备案要求

### 1. ✅ 必须进行：人脸核验

**要求说明**：
- 备案时必须由负责人**本人**进行人脸识别核验
- 通过微信扫码完成
- 必须在光线充足的环境下进行

**准备事项**：
```
✓ 确认负责人本人可配合
✓ 确认负责人有微信号
✓ 提前告知需要人脸核验
✓ 预留足够的操作时间
```

---

### 2. ❌ 不需要：手持身份证照片

**说明**：
- 企业主体不需要提供手持身份证照片
- 只需要：身份证正反面 + 证件照

**注意**：
- 个人主体可能仍需要手持身份证照片
- 本文档仅适用于企业主体

---

### 3. ⏰ 强制要求：必须完成备案

**政策要求**：
- **新小程序**：必须先完成备案才能上架
- **所有小程序**：未备案无法正常运营
- **备案周期**：建议预留7-15个工作日

---

## 📋 备案资料清单

### 企业小程序备案所需资料（2025年）

```
□ 主办单位通信地址（详细到门牌号）
□ 负责人身份证正反面
□ 负责人证件照（蓝底/白底）
□ 负责人本人完成人脸核验 ⭐ 必须
□ 负责人微信号（用于人脸核验）⭐ 必须
□ 核验单（打印签字盖章后扫描）
```

## ⚠️ 常见问题

### Q1: 人脸核验必须本人完成吗？
```
A: 是的，必须由负责人本人完成，无法代办。
   系统会将人脸识别结果与身份证照片进行比对。
```

### Q2: 人脸核验失败怎么办？
```
A: 常见原因：
   ❌ 光线不足 → 在明亮环境下重试
   ❌ 遮挡面部 → 摘掉眼镜、口罩等
   ❌ 身份证过期 → 更新身份证
   ❌ 照片与本人差异大 → 重新拍摄证件照
```

### Q3: 企业主体真的不需要手持身份证照片了吗？
```
A: 是的，根据现行政策：
   ✅ 企业主体：不需要手持身份证照片
   ❌ 个人主体：可能仍需要（本文档不涉及）
```

### Q4: 备案审核需要多长时间？
```
A: 一般情况：
   - 资料提交：1-2天
   - 管局审核：3-7个工作日
   - 总计：建议预留7-15个工作日
   
   注意：各地管局审核速度可能不同
```

### Q5: 负责人必须是法人吗？
```
A: 不是。负责人可以是：
   ✅ 法定代表人
   ✅ 企业员工
   ✅ 其他授权人员
   
   但必须：
   ✓ 提供真实身份信息
   ✓ 本人可配合人脸核验
   ✓ 承担相应责任
```

## 📝 资料准备建议（2025年版）

```

```

```
□ 确认负责人随时可配合
```

## 🎯 重点提示

### ⭐ 三个最容易出问题的地方

1. **管理员微信未绑定银行卡**
   - 注册服务号时就需要
   - 很多人容易忽略
   - 请务必提前检查

2. **负责人不在场无法人脸核验**
   - 备案时必须本人操作
   - 无法代办或延后
   - 建议提前协调好时间

3. **核验单签字或盖章不规范**
   - 必须法人亲笔签字
   - 必须加盖企业公章（红章）
   - 扫描件必须清晰完整
