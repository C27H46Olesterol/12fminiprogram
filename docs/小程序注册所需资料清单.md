# 企业微信小程序注册所需资料清单

> 📅 **更新时间**：2025年10月25日  
> 🎯 **适用对象**：企业主体（含个体工商户）  
> ⭐ **推荐方案**：服务号认证 + 关联小程序

---

## 📑 资料清单速查

### ✅ 注册服务号需要
```
□ 未注册过的邮箱
□ 营业执照（彩色扫描件，≤2MB）
□ 管理员身份证正反面（≤2MB）
□ 管理员微信号（已实名+绑定银行卡）
□ 管理员手机号（能接收验证码）
```

### ✅ 服务号认证需要
```
□ 营业执照扫描件
□ 法人身份证正反面
□ 运营者身份证正反面（可以不是法人）
□ 运营者手机号、邮箱
□ 对公账户信息（开户行、账号）
□ 认证费用：300元
```

### ✅ 注册小程序需要
```
□ 小程序名称（2-30字符）
□ 小程序简介（4-120字符）
□ 小程序图标（144×144px）
□ 服务类目选择
```

### ✅ 小程序备案需要
```
□ 主办单位通信地址（详细到门牌号）
□ 负责人身份证正反面
□ 负责人证件照（蓝底/白底）
□ 负责人本人进行人脸核验（手机操作）⭐
□ 核验单（打印签字盖章后扫描）
```

⚠️ **2024年新要求**：
- 需要负责人本人使用微信进行**人脸识别核验**
- 企业主体通常不需要手持身份证照片

## 一、服务号注册资料

### 1. 邮箱账号 ⭐

| 项目 | 要求 |
|------|------|
| **邮箱地址** | ✅ 未注册过微信公众平台的邮箱<br>✅ 能正常接收邮件<br>❌ 一个邮箱只能注册一个公众号 |

**推荐邮箱**：`service@company.com`、`mp@company.com`

---

### 2. 营业执照 ⭐⭐⭐

| 项目 | 要求 |
|------|------|
| **格式** | JPG 或 PNG |
| **大小** | ≤ 2MB |
| **要求** | ✅ 彩色扫描件或高清照片<br>✅ 清晰完整，四角可见<br>✅ 必须在有效期内<br>❌ 不能有水印、遮挡、反光 |

**必需信息**：
- 企业名称（完整）
- 统一社会信用代码（18位）
- 法定代表人姓名
- 注册地址
- 经营范围

---

### 3. 管理员信息 ⭐⭐

| 项目 | 要求 |
|------|------|
| **姓名** | 真实姓名 |
| **身份证号** | 18位，在有效期内 |
| **身份证照片** | 正反面，JPG/PNG，≤2MB，清晰无反光 |
| **微信号** | ⭐ **必须已实名认证 + 绑定银行卡** |
| **手机号** | 能接收验证码 |

**⚠️ 重要**：
- 管理员微信必须绑定银行卡！
- 如果此人也是备案负责人，后续还需要进行人脸核验

**绑定银行卡路径**：
```
微信 → 我 → 服务 → 钱包 → 银行卡 → 添加银行卡
```

### 1. 企业资料

| 项目 | 要求 |
|------|------|

| 企业名称 | 与营业执照完全一致 |
| 统一社会信用代码 | 18位代码 |

---

### 2. 法人信息

| 项目 | 要求 |
|------|------|
| 法人姓名 | 与营业执照一致 |
| 身份证号 | 18位，在有效期内 |
| 身份证照片 | 正反面，清晰无反光，≤2MB |

---

### 3. 运营者信息（可以不是法人）

| 项目 | 要求 |
|------|------|
| 运营者姓名 | 真实姓名 |
| 身份证号 | 18位，在有效期内 |
| 身份证照片 | 正反面，清晰无反光，≤2MB |
| 手机号 | 能接收验证码 |
| 邮箱 | 能接收邮件 |

---

### 4. 对公账户信息

| 项目 | 要求 |
|------|------|
| 开户银行 | 如：中国工商银行XX支行 |
| 银行账号 | ✅ 企业对公账户<br>⚠️ 个体工商户可用法人银行卡 |
| 开户名称 | 与企业名称完全一致 |

**用途**：接收腾讯打款验证（0.01-0.99元随机金额）

### 1. 小程序基本信息

| 项目 | 要求 | 示例 |

| **简介** | 4-120个字符 | 提供家电维修报修、工单管理服务 |

### 2. 关联说明

**从服务号后台注册小程序**：
```
✅ 自动复用服务号的企业资料
✅ 无需重复提交营业执照、身份证等
✅ 无需支付认证费
✅ 自动获得认证状态
```

## 四、小程序备案资料

> ⏰ **备案时间要求**：
> - 新小程序：必须先完成备案才能上架
> - 所有小程序：未备案无法正常运营
> - 建议预留**7-15个工作日**完成备案流程

### 1. 主体信息

| 项目 | 要求 |
|------|------|
| 主办单位名称 | 与营业执照一致 |
| 统一社会信用代码 | 18位代码 |
| 主办单位通信地址 | **详细到门牌号**（如：XX省XX市XX区XX路XX号XX室）|
| 投资人或上级主管单位 | 一般填企业名称 |

---

### 2. 负责人信息

| 项目 | 要求 |
|------|------|
| 负责人姓名 | 真实姓名 |
| 身份证号 | 18位，在有效期内 |
| 手机号 | 能接收验证码 |
| 邮箱 | 能接收邮件 |

---

### 3. 负责人照片

#### 📸 身份证照片
```
✅ 正反面高清照片
✅ JPG/PNG，≤2MB
✅ 清晰无反光、遮挡
```

#### 📸 证件照
```
✅ 正面免冠照
✅ 蓝底或白底
✅ 清晰可见五官
✅ 近期照片（3个月内）
✅ JPG/PNG，≤2MB
```

⚠️ **说明**：企业主体通常只需要以上两种照片，一般不需要手持身份证照片

---

### 4. 人脸核验 ⭐ 新增要求

**操作流程**：
```
1. 备案时系统会生成二维码
2. 负责人使用微信扫码
3. 按照提示完成人脸识别
4. 系统自动完成核验
```

**⚠️ 重要提示**：
- 必须由负责人**本人**进行
- 需要在光线充足的环境下
- 建议提前准备好负责人的微信号
- 人脸核验有时间限制，请及时完成

---

### 5. 核验单

**处理流程**：
```
1. 系统自动生成核验单PDF
2. 下载并打印
3. 法定代表人签字
4. 加盖企业公章（红章）
5. 彩色扫描成电子版
6. 上传到备案系统
```

## 五、资料准备技巧

### 📸 照片拍摄建议

**身份证拍摄**：
```
✅ 平铺在白色背景上
✅ 自然光或明亮环境
✅ 垂直拍摄，四角完整
✅ 避免反光、阴影
```

**人物照片拍摄**：
```
✅ 正面免冠，五官清晰
✅ 背景干净（纯色最佳）
✅ 光线充足均匀
✅ 不要过度美颜
```

---

### 📁 文件格式要求

| 文件类型 | 格式 | 大小 | 备注 |
|---------|------|------|------|
| 营业执照 | JPG/PNG | ≤2MB | 彩色扫描件 |
| 身份证 | JPG/PNG | ≤2MB | 清晰无反光 |
| 证件照 | JPG/PNG | ≤2MB | 蓝底/白底 |
| 小程序图标 | JPG/PNG | ≤2MB | 144×144px |
| 核验单 | PDF/JPG | ≤5MB | 彩色扫描 |

---

### ⚠️ 常见错误避免

```
❌ 营业执照过期
❌ 身份证过期
❌ 照片模糊、反光
❌ 信息填写与证件不一致
❌ 微信未实名认证
❌ 微信未绑定银行卡 ⭐（最容易忽略）
❌ 邮箱已被注册
❌ 手机号无法接收验证码
❌ 核验单未签字或未盖章
❌ 人脸核验时负责人不在场 ⭐（必须本人操作）
```

## ❓ 资料相关常见问题

**Q1: 管理员、运营者、负责人可以是同一个人吗？**
```
A: 可以！可以都是法人本人，也可以都是其他同一个人。
```

**Q2: 个体工商户没有对公账户怎么办？**
```
A: 可以使用法人的个人银行卡代替对公账户。
```

**Q3: 营业执照是老版（非三证合一）怎么办？**
```
A: 需要额外提供组织机构代码证。
   建议到工商局更换为新版营业执照（免费）。
```

**Q4: 证件照可以用手机拍吗？**
```
A: 可以，但要求：
   ✅ 蓝底或白底（可用证件照小程序）
   ✅ 正面免冠
   ✅ 五官清晰
   ✅ 符合证件照标准
```

**Q5: 人脸核验失败怎么办？**
```
A: 常见原因和解决方法：
   ❌ 光线不足 → 在明亮环境下重试
   ❌ 遮挡面部 → 摘掉眼镜、口罩等
   ❌ 身份证过期 → 更新身份证后重新上传
   ❌ 照片与本人差异大 → 重新拍摄证件照
```

**Q6: 小程序备案政策有哪些重要变化？**
```
A: 主要变化（2024年起实施）：
   ✅ 新增：人脸核验环节（必须本人完成）
   ✅ 强制要求：未备案小程序无法上架
   ✅ 已上架小程序：必须完成补备案
   ❌ 取消：企业主体不再要求手持身份证照片
   
   注：这些要求自2024年4月起已全面执行
```
