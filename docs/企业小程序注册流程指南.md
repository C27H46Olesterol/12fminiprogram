# 企业微信小程序注册流程指南

> 📅 **更新时间**：2025年10月25日  
> 📋 **适用对象**：企业主体（含个体工商户）  
> ⏱ **总流程时长**：约15-30个工作日  
> ⭐ **推荐方案**：服务号认证 + 关联小程序

---

## 💡 为什么推荐"服务号认证+关联小程序"？

| 对比项 | 服务号认证+关联 ⭐ | 小程序单独认证 |
|-------|------------------|---------------|
| **费用** | 300元/年（一次性）| 300元/年 × 每个小程序 |
| **多小程序** | ✅ 关联多个全免费 | ❌ 每个都要付费 |
| **功能** | ✅ 完整（手机号、支付等）| ✅ 完整（完全一样）|
| **管理** | ✅ 统一管理方便 | ❌ 分散管理 |

**核心优势**：认证一次，多个小程序免费复用，功能无任何差别！

## 📑 操作流程

### 第一步：注册微信服务号（10分钟）

#### 1. 访问注册页面
```
🌐 https://mp.weixin.qq.com/
点击【立即注册】→ 选择【服务号】
```

#### 2. 填写基本信息
- **邮箱**：未注册过的邮箱（作为登录账号）
- **密码**：设置登录密码
- **验证邮箱**：点击邮件中的链接

#### 3. 选择主体类型
- 选择：**企业**（个体工商户也选企业）

#### 4. 填写企业信息
- 企业名称（与营业执照一致）
- 统一社会信用代码（18位）
- 营业执照照片（彩色扫描件）

#### 5. 填写管理员信息
- 管理员姓名、身份证号
- 管理员微信号（**必须已实名+绑定银行卡**）
- 管理员手机号（接收验证码）
- 扫码验证身份

✅ **完成注册**

---

### 第二步：服务号认证（3-7个工作日）

#### 1. 登录服务号后台
```
🌐 https://mp.weixin.qq.com/
使用管理员微信扫码登录
```

#### 2. 进入认证页面
```
左侧菜单 → 设置 → 微信认证 → 【开通】
```

#### 3. 填写认证资料

**企业信息**
```
- 企业名称：与营业执照一致
- 统一社会信用代码：18位
- 营业执照照片：清晰彩色扫描件（JPG/PNG，≤2MB）
```

**法人信息**
```
- 法人姓名、身份证号
- 法人身份证照片：正反面（清晰无反光）
```

**运营者信息**（可以不是法人）
```
- 运营者姓名、身份证号
- 运营者身份证照片：正反面
- 运营者手机号、邮箱
```

**对公账户**
```
- 开户银行：如"中国工商银行XX支行"
- 银行账号：对公账户（个体工商户可用法人银行卡）
- 开户名称：与企业名称一致
```

#### 4. 支付认证费
```
💰 费用：300元（有效期1年）
💳 支付方式：微信支付 或 对公转账
```

#### 5. 等待审核
```
对公打款验证：1-3个工作日（会打款0.01-0.99元）
  → 登录网银查看打款金额
  → 在认证页面填写金额验证

人工审核：3-5个工作日
  → 审核通过后短信+公众平台通知
```

✅ **认证完成，获得"已认证"标识**

---

### 第三步：注册并关联小程序（10分钟）

#### 方式一：从服务号后台直接注册小程序（推荐）

**1. 进入小程序管理**
```
服务号后台 → 小程序管理 → 快速注册并认证小程序
```

**2. 快速注册**
```
- 使用已认证服务号资质快速注册
- 无需重复提交企业资料
- 无需支付费用
- 复用服务号的认证状态
```

**3. 填写小程序信息**
```
- 小程序名称：2-30个字符（如：XX售后平台）
- 小程序简介：简要说明功能（4-120字）
- 小程序图标：144×144px，JPG/PNG，≤2MB
- 服务类目：选择对应行业
```

✅ **完成！小程序自动获得认证状态，无需额外付费**

---

#### 方式二：先单独注册小程序，再关联到服务号

**1. 注册小程序**
```
🌐 https://mp.weixin.qq.com/
立即注册 → 选择【小程序】→ 填写信息
（使用不同邮箱，流程同注册服务号）
```

**2. 关联到服务号**
```
服务号后台 → 小程序管理 → 关联小程序
  → 输入小程序AppID
  → 小程序管理员确认关联
```

**3. 复用认证**
```
关联成功后，小程序自动继承服务号的认证状态
无需单独认证，无需支付300元
```

✅ **完成！小程序免费获得认证**

---

### 第四步：小程序备案（7-20个工作日）

#### 1. 登录小程序后台
```
🌐 https://mp.weixin.qq.com/
左侧菜单 → 设置 → 基本设置 → 【小程序备案】
```

#### 2. 填写备案信息

**主体信息**
```
- 主办单位名称：与营业执照一致
- 统一社会信用代码
- 主办单位通信地址：详细到门牌号
- 投资人或上级主管单位：一般填企业名称
```

**负责人信息**
```
- 负责人姓名、身份证号
- 负责人手机号、邮箱
- 负责人身份证照片：正反面
- 负责人证件照：蓝底或白底
- 负责人手持身份证照片：本人手持身份证正面
```

**小程序信息**
```
- 小程序名称
- 服务内容类型
- 服务内容描述
```

#### 3. 上传备案材料
```
✅ 营业执照扫描件
✅ 负责人身份证正反面
✅ 负责人证件照（蓝底/白底）
✅ 负责人手持身份证照片
✅ 核验单（系统生成 → 打印 → 签字盖章 → 扫描上传）
```

#### 4. 提交审核
```
微信初审：1-2个工作日
管局审核：7-20个工作日
审核通过：获得备案号（ICP备XXXXXXXX号）
```

✅ **备案完成，小程序可以正式上线**

## ❓ 常见问题

**Q1: 服务号和订阅号有什么区别？**
```
A: 必须注册服务号！
   - 服务号：可以认证，可以关联小程序 ✅
   - 订阅号：企业订阅号可以认证，但功能受限 ⚠️
```

**Q2: 服务号认证后，小程序真的免费吗？**
```
A: 是的！服务号认证后（300元/年），可以：
   ✅ 免费注册并关联小程序
   ✅ 小程序自动获得认证状态
   ✅ 获取手机号、微信支付等功能完全可用
   ✅ 关联多个小程序全部免费
```

**Q3: 功能和单独认证小程序有差别吗？**
```
A: 完全没有差别！
   ✅ 获取手机号：可用
   ✅ 微信支付：可用
   ✅ 所有高级接口：可用
   ✅ 已认证标识：有
   两种方式认证后的权限100%相同
```

**Q4: 一个服务号可以关联多少个小程序？**
```
A: 最多可以关联50个小程序
   而且全部免费复用认证状态！
```

**Q5: 已经单独注册的小程序，能改用这个方案吗？**
```
A: 可以！步骤：
   1. 注册并认证服务号（300元/年）
   2. 服务号后台关联已有的小程序
   3. 小程序自动获得认证状态
   
   注意：如果小程序已经单独认证，
   下次续费时可以不续费小程序，只续费服务号即可
```

**Q6: 个体工商户可以用这个方案吗？**
```
A: 可以！流程完全一样
   区别：对公账户验证时可以使用法人个人银行卡
```

**Q7: 必须先认证服务号才能关联小程序吗？**
```
A: 是的！流程顺序：
   1. 注册服务号
   2. 服务号认证（支付300元）
   3. 认证通过后，注册/关联小程序
   4. 小程序免费获得认证
```

**Q8: 服务号认证到期后，小程序也会失效吗？**
```
A: 是的！服务号认证到期后：
   - 服务号失去认证状态
   - 关联的小程序也同时失去认证状态
   - 小程序的获取手机号等功能会失效
   
   需要在到期前续费（300元/年）
```
