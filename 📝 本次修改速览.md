# 📝 本次修改速览

## ✅ 已完成的修改

### 1️⃣ 删除"待发件"筛选标签
- **位置**: 主管 → 待处理问题页面
- **结果**: 只保留4个标签（待分配、处理中、待返件、待完成）

### 2️⃣ 修复"问题不存在"错误
- **问题**: 点击"立即评价"显示"加载失败"
- **原因**: 工单ID格式是 `M-251101-006`，但云函数只识别 `ISSUE_` 开头的ID
- **修复**: 云函数现在可以识别 `M-` 和 `ISSUE_` 开头的ID

### 3️⃣ 已解决问题列表显示编号
- **位置**: 主管 → 已解决问题页面
- **新增**: 每个问题都显示问题编号（如 `M-251101-006`）
- **样式**: 浅灰色背景 + 蓝色左边框 + 等宽字体
